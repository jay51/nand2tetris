### To run

- export `./lib` to `LD_LIBRARY_PATH`
- compile with `make` (gcc -L <path-to-so> -l<hashtable.so> -I ./includes source files)
- run with `assembler Add.asm`

> This may not work with windows line terminating `\r\n`
